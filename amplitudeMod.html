MESSAGE SIGNAL

Here we create a message signal for our simulation Amplitude of modulating signal

Am=5;
% Frequency of modulating signal
fa=2000;
% Time period of modulating signal
Ta=1/fa;
% Total time for simulation
t=0:Ta/999:6*Ta;
% Eqation of modulating signal
ym=Am*sin(2*pi*fa*t);

Graphical representation of Modulating signal

This plots the message signal also called as the modulating signal

subplot(2,2,1);
plot(t,ym), grid on;
t1=title ( '  Modulating Signal   ');
t1.Color = 'blue';
xlabel ( ' time(sec) ');
ylabel (' Amplitude(volt)   ');

CARRIER

We create our carrier wave for amplitude modulation Amplitude of carrier signal [ where, modulation Index (m)=Am/Ac ]

Ac=Am/m;
% Frequency of carrier signal
fc=fa*10;
% Time period of carrier signal
Tc=1/fc;
% Eqation of carrier signal
yc=Ac*sin(2*pi*fc*t);

Graphical representation of carrier signal

This section plots our carrier wave, here notice the difference as carrier wave has a higher frequency it has more peaks and troughs than the message signal

subplot(2,2,2);
plot(t,yc,'r'), grid on;
t2=title ( '  Carrier Signal   ');
t2.Color = 'red';
xlabel ( ' time(sec) ');
ylabel (' Amplitude(volt)   ');

AM Modulation

To create this simulated Amplitude Modulation we create this AM wave using the equation given below Equation of Amplitude modulated signal

y=Ac*(1+m*sin(2*pi*fa*t)).*sin(2*pi*fc*t);

Graphical representation of AM signal

Finlly we plot the AM wave (Yea!!)

subplot(2,2,[3,4]);

t3=title ( '  Amplitude Modulated signal  ');
